## Superstore Sales Dashboard (Power BI)

**Description:**
Interactive Power BI dashboard analyzing sales, profit, quantity, and delivery performance using the Superstore dataset.

**Key Features:**
- KPI cards for Sales, Profit, Quantity, Avg Delivery Days
- Sales analysis by region, category, and segment
- Top and bottom performing products
- Dynamic slicers for year, market, and category

**Tools Used:**
Power BI, DAX
